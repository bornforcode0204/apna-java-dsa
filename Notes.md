# ▶ if-else Statement:

```java
if(condition){
        code
    }else{
        code
    }
```

# ▶ else if Statement:

```java
if(condition 1){
        code
    }else if(condition 2){
        code
    }else{
        code
    }
```

# ▶ Ternary Operator:

                    Ternary Operator is an aletrnate way to write if-else.
                    unary --> 1 operand
                    binary --> 2 operands
                    ternary --> 3 operands

```java
    variable = condition? statement1 : statement2;

    boolean larger = (5>3)? 5(true):3(false);

    String type = (5%2 == 0)? "even" : "odd";
```

# ▶ Switch Statement:

```java
switch(variable){
    case 1:
            code
            break;
    case 2:
            code
            break;
    case 3:
            code
            break;
    default:
            stmt
            break;(optional)
}
```
